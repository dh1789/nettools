# PRD: 사이드바 네비게이션

## 개요
NetTools 프로젝트에 반응형 사이드바 네비게이션을 추가하여 도구 간 자유로운 이동을 지원한다.

## 목표
- 데스크톱: 왼쪽 고정 사이드바로 카테고리별 도구 목록 표시
- 모바일: 햄버거 메뉴로 접히는 오버레이 사이드바
- 현재 페이지 하이라이트
- 다국어(ko/en) 완전 지원
- 카테고리 접기/펼치기

## 사용자 시나리오

### 시나리오 1: 데스크톱에서 도구 탐색
1. 사용자가 Subnet Calculator 페이지에 진입
2. 왼쪽 사이드바에 카테고리별 도구 목록이 보임
3. "MAC 주소 OUI 조회기" 클릭 → 해당 페이지로 이동
4. 사이드바에서 현재 도구가 하이라이트됨

### 시나리오 2: 모바일에서 도구 탐색
1. 모바일 브라우저에서 접속
2. 사이드바 숨김, 헤더에 햄버거 메뉴(☰) 버튼 표시
3. 햄버거 클릭 → 왼쪽에서 오버레이 사이드바 슬라이드
4. 도구 선택 → 이동 후 사이드바 자동 닫힘

### 시나리오 3: 홈페이지에서 사이드바
1. 홈페이지(/) 진입 시에도 사이드바 표시
2. 홈 링크가 상단에 위치
3. 카테고리 접기/펼치기 가능

## 기능 요구사항
- TOOLS, CATEGORIES 데이터로 자동 메뉴 생성
- 현재 URL 기반 active 상태 표시
- 카테고리 토글(접기/펼치기)
- 모바일 오버레이 + 배경 dimming
- 반응형 breakpoint: 768px
- 다국어 지원 (useLocale 통합)
- 도구 없는 카테고리는 숨김

## 비기능 요구사항
- 정적 빌드(output: export) 호환
- 기존 inline style 패턴 유지
- CSS 변수(--border, --surface 등) 활용
- 번들 사이즈 최소화 (외부 라이브러리 금지)

## 기술 스택
- Next.js 15 (App Router, static export)
- React 19
- TypeScript 5.7
- Inline Styles + CSS Variables

## 성공 지표
- 모든 도구 페이지에서 사이드바 렌더링 정상
- 768px 미만에서 햄버거 전환 동작
- 현재 페이지 하이라이트 정확
- 한국어/영어 전환 시 사이드바 텍스트 즉시 변경
- 카테고리 접기/펼치기 정상 동작
